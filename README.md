# Git and GitHub Final Project

## Project Description
This repository contains the final project for the Git and GitHub course. It demonstrates proficiency in using GitHub's web interface and Git command-line tools.

## Project Title
**Simple Interest Calculator**

## Description
A simple interest calculator built with Bash scripting that allows users to calculate simple interest based on principal amount, rate of interest, and time period.

## Features
- Calculate simple interest using the formula: SI = (P × R × T) / 100
- Interactive command-line interface
- Input validation
- Clear output formatting

## Usage
Run the simple interest calculator:
```bash
bash simple-interest.sh
```
